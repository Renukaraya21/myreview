# review.in

Teacher-student review platform built with React and Vite.

## Quick start

1. `npm install`
2. Copy `.env.example` to `.env` and adjust values.
3. `npm run dev`

## Demo credentials (mock auth)

- Teacher: `teacher@review.in` / `Teach123!`
- Student: `student@review.in` / `Study123!`
